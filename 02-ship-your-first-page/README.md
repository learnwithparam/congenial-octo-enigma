# 02 -- Ship Your First Page

Build a Beautiful Frontend with Next.js, Tailwind, and Mock Data.

## What You'll Learn

How to build a polished, responsive frontend application using Next.js 15,
Tailwind CSS v4, and mock data -- from project setup through deployment.

## Prerequisites

- Basic HTML, CSS, and JavaScript knowledge
- Node.js installed (v18+)
- Basic understanding of how web applications work

### Quick Start

```bash
make setup    # Install dependencies
make dev      # Start Next.js dev server at http://localhost:3000
```
