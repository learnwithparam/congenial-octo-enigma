## What You'll Learn

How to use AI coding tools effectively in your development workflow. You will set up Claude Code and Cursor, learn prompt engineering for code generation, use AI for code review, refactoring, and debugging, and build repeatable workflows that make you faster.

## Prerequisites

- Comfortable writing code in any language
- A text editor or IDE installed
- An Anthropic API key (for Claude Code)

### Quick Start

```bash
make setup    # Install dependencies
make dev      # Start development
```
